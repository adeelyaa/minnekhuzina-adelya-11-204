# Release Notes

## v1.0
- Реализован краулер
- Скачивание 100 страниц
- Сохранение HTML без обработки
- Генерация index.txt

## v2.0
- Добавлена токенизация текстов
- Удалены дубликаты, стоп-слова и мусорные токены
- Реализована лемматизация
- Токены сохраняются отдельно для каждого документа (tokens_by_page/)
- Леммы сохраняются отдельно для каждого документа (lemmas_by_page/)

## v3.0
- Реализовано построение инвертированного индекса
- Добавлен файл inverted_index.txt
- Реализован булев поиск с операторами AND, OR, NOT
- Добавлена поддержка сложных запросов со скобками

## v4.0
- Добавлен скрипт `calc_tfidf.py`
- Реализован подсчёт TF, IDF и TF-IDF для терминов
- Реализован подсчёт TF, IDF и TF-IDF для лемм
- Сгенерированы папки `term_tfidf` и `lemma_tfidf` с результатами для каждого документа

## v5.0
- Реализован векторный поиск
- Добавлено ранжирование документов по косинусной близости
- Поддержка поиска по строковому запросу
